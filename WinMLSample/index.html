This sample demostrates using Windows ML apis for model inference from a JavaScript Web App.
In this sample, we are using the tinyyolo object identification ONNX model to detect a list of 20 objects from a video frame/photo. Objects identified using this model include aeroplane, bicycle, bird, boat, bottle, bus, car, cat, chair, cow, diningtable, dog, horse, motorbike, person, pottedplant, sheep, sofa, train, tvmonitor
